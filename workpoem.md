# work

I've done a lot of work.
I've done work to make me money,
I've done work for my resume,
I've done a lot of homework.
I've done mental, physical, emotional, social, and spiritual work.
I've also done work that might not seem like work.
Work that creates memories,
scratches creative itches,
fulfills favors,
drives new desires,
and revives childhood obsessions.
I've found myself enjoying and testing different ways of working for a while now.
What do I want to do for work?
I guess I'm still working on it.
